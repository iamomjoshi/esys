Timer Functions

1. Timer Functions Overview

Arduino provides built-in timer functions that allow precise time tracking without blocking code execution.

2. Common Timer Functions

A. millis() - Get Elapsed Time in Milliseconds

Returns the number of milliseconds since the Arduino board started running the current program.

unsigned long timeElapsed = millis();
B. micros() - Get Elapsed Time in Microseconds

Returns the number of microseconds since the program started running.

unsigned long timeElapsedMicro = micros();
C. delay() - Pause Execution in Milliseconds

Pauses the program for the specified number of milliseconds.

delay(1000); // Waits for 1 second
D. delayMicroseconds() - Pause Execution in Microseconds

Pauses the program for the specified number of microseconds.

delayMicroseconds(100); // Waits for 100 microseconds

Arduino Communication Functions

1. Introduction to Communication in Arduino

Communication is essential for interfacing Arduino with other devices, sensors, and computers. Arduino supports multiple communication protocols, including Serial, I2C, and SPI.

2. Serial Communication

Serial communication is used to transfer data between Arduino and external devices, such as computers or other microcontrollers, using the UART protocol.

A. Serial.begin() - Initialize Serial Communication

Sets the baud rate for serial communication.

Serial.begin(9600); // Initializes serial at 9600 baud rate
B. Serial.print() & Serial.println() - Sending Data

Sends data over the serial port.

Serial.print("Hello, Arduino!");
Serial.println("This is a new line");
C. Serial.read() & Serial.available() - Receiving Data

Reads incoming serial data.

if (Serial.available() > 0) {
    char data = Serial.read();
    Serial.print("Received: ");
    Serial.println(data);
}

3. I2C Communication

I2C (Inter-Integrated Circuit) is a two-wire protocol used for communication between multiple devices.

A. Wire.begin() - Initialize I2C

Starts the I2C communication.

#include <Wire.h>
Wire.begin();
B. Wire.requestFrom() - Read Data from a Device

Requests data from an I2C slave device.

Wire.requestFrom(8, 6); // Request 6 bytes from device at address 8
C. Wire.write() & Wire.read() - Sending and Receiving Data

Sends and receives data over I2C.

Wire.beginTransmission(8);
Wire.write("Hello");
Wire.endTransmission();

4. SPI Communication

SPI (Serial Peripheral Interface) is a high-speed communication protocol used for interfacing with sensors and peripherals.

A. SPI.begin() - Initialize SPI Communication

Starts SPI communication.

#include <SPI.h>
SPI.begin();
B. SPI.transfer() - Send and Receive Data

Transfers data over SPI.

byte data = SPI.transfer(0xFF); // Send 0xFF and receive response

5. Summary Table

Function Purpose Example

Serial.print() Send Data Over Serial Serial.print("Hello");
Wire.begin() Initialize I2C Communication Wire.begin();
SPI.begin() Initialize SPI Communication SPI.begin();

Arduino Interrupt Functions

1. Introduction to Interrupts in Arduino

Interrupts allow Arduino to respond to events immediately, without continuously checking conditions in the main loop. When an interrupt occurs, the current code execution is paused, and a special function (Interrupt Service Routine) is executed.

2. Types of Interrupts

Arduino supports two types of interrupts:

  • External Interrupts: Triggered by external signals on specific pins.
  • Pin Change Interrupts: Triggered when a digital pin changes state.

3. External Interrupt Functions

External interrupts are triggered on specific pins (e.g., pin 2 and pin 3 on Arduino Uno).

A. attachInterrupt() - Enable an Interrupt

Attaches an interrupt to a specified pin.

attachInterrupt(digitalPinToInterrupt(2), myISR, RISING);

This line sets pin 2 to trigger the myISR function on a rising edge.

B. detachInterrupt() - Disable an Interrupt

Detaches an interrupt from a pin, stopping it from triggering.

detachInterrupt(digitalPinToInterrupt(2));

4. Interrupt Service Routine (ISR)

The ISR is the function that runs when an interrupt occurs. It should be as short as possible and avoid using functions like delay() or Serial.print().

void myISR() {
    digitalWrite(LED_BUILTIN, !digitalRead(LED_BUILTIN)); // Toggle LED
}

5. Pin Change Interrupts

Pin change interrupts are available on all digital pins, but they require direct register manipulation.

A. Enabling Pin Change Interrupts
PCICR |= (1 << PCIE0);  // Enable Pin Change Interrupts for PCINT[7:0]
PCMSK0 |= (1 << PCINT4);  // Enable interrupt on pin 4
B. Pin Change Interrupt Service Routine
ISR(PCINT0_vect) {
    Serial.println("Pin state changed!");
}

Arduino Math Functions

1. Introduction to Math Functions

Arduino provides various mathematical functions to perform calculations efficiently. These functions help in manipulating numbers, performing trigonometry, rounding values, and more.

2. Basic Math Functions

A. abs() - Absolute Value

Returns the absolute (positive) value of a number.

int x = -10;
int y = abs(x); // y = 10
B. constrain() - Limit a Value

Constrains a value within a given range.

int val = constrain(150, 0, 100); // val = 100
C. max() and min() - Maximum and Minimum

Returns the maximum or minimum value between two numbers.

int a = max(10, 20); // a = 20
int b = min(10, 20); // b = 10

3. Rounding Functions

A. round() - Round to Nearest Integer

Rounds a floating-point number to the nearest whole number.

float x = 4.7;
int y = round(x); // y = 5
B. floor() and ceil() - Round Down or Up

floor() rounds down, and ceil() rounds up.

float x = 4.7;
int a = floor(x); // a = 4
int b = ceil(x);  // b = 5

4. Trigonometric Functions

A. sin(), cos(), and tan()

Returns the sine, cosine, or tangent of an angle (in radians).

float angle = PI / 4;
float s = sin(angle);
float c = cos(angle);
float t = tan(angle);
B. radians() and degrees()

Converts between degrees and radians.

float rad = radians(180); // rad = PI
float deg = degrees(PI); // deg = 180

5. Random Number Functions

A. random() - Generate Random Numbers

Generates a random number within a specified range.

int num = random(1, 100); // Random number between 1 and 99
B. randomSeed() - Set Random Seed

Initializes the random number generator with a seed.

randomSeed(analogRead(0));

Arduino Programming Language Reference

1. Introduction

Arduino programming is based on C/C++ and provides a set of built-in functions to interact with hardware easily.

2. Structure

A. setup()

Runs once when the Arduino is powered on or reset.

void setup() {
    Serial.begin(9600); // Initialize Serial Communication
}
B. loop()

Runs continuously after setup() has completed.

void loop() {
    Serial.println("Hello, Arduino!");
    delay(1000); // Wait for 1 second
}

3. Data Types

A. Integer Types

Used for whole numbers.

int a = 10;
long b = 100000L;
byte c = 255;
B. Floating Point

Used for decimal numbers.

float pi = 3.14;
double precise = 3.141592653;

4. Control Structures

A. if-else

Used for conditional execution.

if (a > 5) {
    Serial.println("a is greater than 5");
} else {
    Serial.println("a is 5 or less");
}
B. for Loop

Repeats a block of code a specific number of times.

for (int i = 0; i < 10; i++) {
    Serial.println(i);
}
C. while Loop

Repeats a block of code while a condition is true.

int count = 0;
while (count < 5) {
    Serial.println(count);
    count++;
}

5. Functions

A. Defining Functions

Functions allow code reuse and organization.

int add(int x, int y) {
    return x + y;
}
B. Calling Functions

Use function calls to execute defined functions.

int sum = add(5, 10);
Serial.println(sum);
